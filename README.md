# Mina zkApp: Anaon Dao Voting

This template uses TypeScript.

## How to setup

```sh
npm install
```

## How to build

```sh
npm run build && node build/src/main.js
```

## Deploy to Berkley testnet

```sh
zk config
zk deploy berkley
npm run build && node build/src/main.js berkley
```
